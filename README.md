## About

This repository documents my progression from Python fundamentals toward data. Each folder contains a focused exercise or mini-project targeting a specific concept. The goal is consistency.

---

## Tools & Technologies

- **Python** — core scripting language
- **pandas** — data manipulation and transformation
- **SQLAlchemy** — database interaction
- **SQLite / PostgreSQL** — local and cloud databases
- **SQL** — querying and aggregating data

---

## How to Use

Each folder is self-contained. To run any script:

```bash
# Install dependencies
pip install -r requirements.txt

# Run a pipeline (sample)
python 01_etl_basics/pipeline.py
```
